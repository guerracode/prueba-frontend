## Sobre el proyecto

El objetivo fue recrear una imagen muestra que tiene graficas y datos. </br>
Se uso PieChart y AreaChart para recrear las graficas.

### Construido con

Estas son las principales tecnologias usadas en el proyecto:

- JavaScript
- React
- Recharts
- Webpack
- Babel

## Probar el proyecto

Se le hizo deploy al proyecto, por lo tanto se puede visualizar accediendo a esta [URL](https://prueba-frontend-delta.vercel.app/).</br>

### Correr en Local

- Descargar o clonar el proyecto.
- Instalar las dependencias con:</br>
  `npm install`
- Correr el proyecto con: </br> `npm run start`
- Se abrira el navegador con la pagina.
